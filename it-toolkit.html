Markdown — Per Documenti e Testi

Markdown è un modo super semplice per formattare il testo. È quello che l'AI usa per formattare le sue risposte — intestazioni, testo in grassetto, elenchi puntati, tabelle. È pulito, privo di formattazione inutile, e facile da leggere e modificare. Quando chiedi all'AI di creare documenti, chiedi il formato Markdown — poi incollalo in Google Docs o nel tuo word processor.

# Titolo del Mio Report ## Introduzione Questo è un paragrafo con **grassetto** e *corsivo*. ### Punti Chiave - Primo punto importante - Secondo punto importante - Terzo punto importante | Paese | Capitale | |-----------|-------------| | Francia | Parigi | | Giappone | Tokyo | | Brasile | Brasilia |

Come Usare Markdown con Google Docs

Passaggio 0: Preparazione

Per prima cosa, abilita il supporto nativo Markdown in Google Docs: vai su Strumenti → Preferenze e seleziona «Attiva Markdown». Questo sblocca funzioni come Incolla da Markdown e la formattazione automatica Markdown.

Per un uso avanzato di Markdown, puoi opzionalmente installare il plugin Docs to Markdown dal Google Workspace Marketplace. Tuttavia, il supporto nativo Markdown è più che sufficiente per la maggior parte degli utenti e i nostri scenari.

Passaggio 1: Chiedi all'AI per Markdown

Quando chiedi all'AI di scrivere un documento, aggiungi: "Per favore, restituisci in formato Markdown". Il risultato sarà un testo pulito e ben strutturato senza formattazione disordinata.

Passaggio 2: Incolla in Google Docs

Copia il testo Markdown dal tuo strumento AI, poi in Google Docs usa Modifica → Incolla da Markdown (o clic destro e seleziona «Incolla da Markdown»). Google Docs converterà automaticamente il Markdown in testo splendidamente formattato — intestazioni, grassetto, elenchi, link e altro verranno renderizzati correttamente.

Nota: L'opzione Incolla da Markdown è disponibile solo se hai completato il Passaggio 0 (abilitazione di Markdown nelle Preferenze). Se non la vedi, torna su Strumenti → Preferenze e assicurati che «Attiva Markdown» sia selezionato.

Passaggio 3: Scarica Google Docs come Markdown

Google Docs può esportare direttamente in Markdown! Vai su File → Scarica → Markdown (.md). Questo ti dà un file Markdown pulito che puoi dare all'AI, usare in GitHub o convertire in qualsiasi altro formato. Funziona anche al contrario — puoi aprire un file .md in Google Docs e verrà visualizzato con la formattazione completa.

Perché Markdown invece del testo normale?

Markdown è pulito — nessuna formattazione nascosta che si rompe quando copi e incolli. È universale — funziona ovunque. Ed è come pensa l'AI — otterrai risultati migliori quando parli la sua lingua.

HTML — Per Contenuti Interattivi

Quando vuoi creare qualcosa che le persone possono aprire in un browser e con cui interagire — come un sito web, un quiz, un grafico interattivo o una presentazione — chiedi all'AI di darti HTML. Non devi imparare l'HTML. L'AI scrive tutto il codice per te. Tu salvi solo il file e lo apri in Chrome o qualsiasi browser.

Cosa Chiedi all'AI

"Crea un quiz interattivo sulla Rivoluzione Francese con 10 domande, un contatore di punteggio e un design colorato. Dammi un singolo file HTML che posso aprire in un browser." L'AI genera TUTTO il codice: - HTML (la struttura) - CSS (il design) - JavaScript (l'interattività) Tu salvi solo come file .html e lo apri! Ecco fatto.

Come Funziona — È Davvero Così Semplice

1. Chiedi all'AI di creare quello che vuoi come "un singolo file HTML" — 2. Copia il codice che l'AI ti dà — 3. Salvalo come qualcosa.html sul tuo computer — 4. Fai doppio clic per aprirlo nel browser. Fatto!

Puoi condividere il file con chiunque — basta aprirlo nel browser. Nessuna app da installare, nessun account da creare. Questo stesso sito web che stai leggendo ora è un file HTML creato con l'AI.

Quando Usare Cosa

Usa Markdown

Saggi, report, appunti, guide di studio, email, documenti che incollerai in Google Docs o Word

Usa HTML

Quiz, presentazioni interattive, giochi, grafici, siti web, qualsiasi cosa gli altri vedranno in un browser

Consiglio Pro

Non devi imparare nessuno dei due formati. Dì semplicemente all'AI cosa vuoi e aggiungi "in Markdown" o "come file HTML" — l'AI fa il resto!

Intuizione Chiave

Più capisci questi formati, più potenti diventano i tuoi risultati con l'AI. Si tratta di comunicazione, non di programmazione.

Mermaid — Diagrammi dal Testo

Mermaid ti permette di creare diagrammi professionali — diagrammi di flusso, timeline, mappe mentali, diagrammi di sequenza — semplicemente scrivendo testo semplice. L'AI scrive il codice Mermaid per te. Descrivi semplicemente il tuo diagramma, incolla il codice nell'Editor Live Mermaid ed esportalo. Puoi anche usare Mermaid per spiegare flussi e processi complessi all'AI.

Perché Mermaid?

Mermaid supporta diagrammi di flusso, diagrammi di sequenza, grafici Gantt, mappe mentali, timeline, grafici a torta, diagrammi di classe e altro. Il codice è basato su testo, quindi l'AI può generarlo e modificarlo facilmente. GitHub renderizza Mermaid nativamente nei file Markdown, e l'Editor Live ti permette di esportare come PNG o SVG.

Consiglio pro: Puoi anche usare Mermaid per spiegare flussi complessi all'AI — incolla un diagramma Mermaid nel tuo prompt così l'AI capisce il tuo processo o architettura a colpo d'occhio.

Pubblica Gratis — GitHub Pages

Hai creato un'app interattiva, un gioco o un sito web straordinario con l'AI. E ora? Condividilo con il mondo gratuitamente usando GitHub Pages. Chiunque abbia il link può usare la tua creazione — nessuna tariffa di hosting, nessuna configurazione server, nessuna conoscenza tecnica necessaria.

Passaggio 1: Crea un Account GitHub Gratuito

Vai su github.com e registrati per un account gratuito. Questa è la tua base per pubblicare siti web e condividere codice.

Passaggio 2: Crea un Nuovo Repository

Clicca "New repository", dagli un nome (questo diventa parte del tuo URL), e assicurati che sia impostato su Public. Pensa a un repository come una cartella per il tuo progetto.

Passaggio 3: Carica il Tuo File HTML

Clicca "Add file" → "Upload files" e trascina il tuo file HTML. Importante: nomina il tuo file principale index.html — questo è quello che GitHub Pages mostrerà quando qualcuno visita il tuo sito.

Passaggio 4: Abilita GitHub Pages

Vai su Settings → Pages del tuo repository. Sotto "Source", seleziona "Deploy from a branch", scegli il branch "main" e clicca Save. In pochi secondi, il tuo sito è online!

Passaggio 5: Condividi il Tuo Link

Il tuo sito sarà live su https://tuousername.github.io/nome-tuo-repo/. Condividi questo link con chiunque — possono aprire la tua creazione interattiva istantaneamente, su qualsiasi dispositivo. Questo stesso sito web è ospitato su GitHub Pages!

100% Gratuito

Nessuna tariffa di hosting, nessuna carta di credito, nessun limite per i progetti personali. GitHub Pages è completamente gratuito.

Aggiornamenti Istantanei

Carica una nuova versione del tuo file e il sito si aggiorna automaticamente. Itera velocemente con l'AI.

Condividi Qualsiasi Cosa

Giochi, quiz, portfolio, storie interattive, visualizzazioni di dati — se l'AI può costruirlo, GitHub Pages può ospitarlo.
