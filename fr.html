Ta boîte à outils IA

Les compétences fondamentales qui rendent tout le reste plus facile

Maîtrise ces trois compétences et tu débloques toute la puissance de l'IA : Markdown pour les documents, HTML pour le contenu interactif, et GitHub Pages pour publier et partager tes créations avec le monde entier — gratuitement.

Markdown — Pour les documents & le texte

Markdown est un moyen ultra simple de formater du texte. C'est ce que l'IA utilise pour formater ses réponses — titres, texte en gras, listes à puces, tableaux. C'est propre, sans formatage inutile, et facile à lire et modifier. Quand tu demandes à l'IA de créer des documents, demande le format Markdown — puis colle-le dans Google Docs ou ton traitement de texte.

Comment utiliser Markdown avec Google Docs

Étape 1 : Demande du Markdown à l'IA

Quand tu demandes à l'IA d'écrire un document, ajoute : « Donne-moi le résultat en format Markdown ». Le résultat sera un texte propre et bien structuré, sans formatage inutile.

Étape 2 : Coller dans Google Docs

Meilleure méthode : Installe le module « Docs to Markdown » depuis le Google Workspace Marketplace. Il convertit le Markdown en texte Google Docs parfaitement formaté en un clic.

Méthode rapide : Copie le Markdown, colle-le, puis utilise Ctrl+Shift+V (Coller sans formatage). Les titres et les listes seront propres et faciles à formater.

Étape 3 : Télécharger Google Docs en Markdown

Google Docs peut exporter directement en Markdown ! Va dans Fichier → Télécharger → Markdown (.md). Tu obtiens un fichier Markdown propre que tu peux donner à l'IA, utiliser sur GitHub, ou convertir dans n'importe quel format. Ça marche aussi dans l'autre sens — tu peux ouvrir un fichier .md dans Google Docs et il s'affichera avec tout le formatage.

Pourquoi Markdown plutôt que du texte normal ?

Markdown est propre — pas de formatage caché qui casse quand tu copies-colles. C'est universel — ça marche partout. Et c'est le langage de l'IA — tu obtiendras de meilleurs résultats en parlant sa langue.

HTML — Pour le contenu interactif

Quand tu veux créer quelque chose que les gens peuvent ouvrir dans un navigateur et avec quoi interagir — comme un site web, un quiz, un graphique interactif ou une présentation — demande à l'IA de te donner du HTML. Tu n'as pas besoin d'apprendre le HTML. L'IA écrit tout le code pour toi. Tu n'as qu'à sauvegarder le fichier et l'ouvrir dans Chrome ou n'importe quel navigateur.

Ce que tu demandes à l'IA

« Crée un quiz interactif sur la Révolution française avec 10 questions, un compteur de score et un design coloré. Donne-moi un seul fichier HTML que je peux ouvrir dans un navigateur. » L'IA génère TOUT le code : - HTML (la structure) - CSS (le design) - JavaScript (l'interactivité) Tu n'as qu'à le sauvegarder en .html et l'ouvrir ! C'est tout.

Comment ça marche — C'est vraiment aussi simple

1. Demande à l'IA de créer ce que tu veux comme « un seul fichier HTML » — 2. Copie le code que l'IA te donne — 3. Sauvegarde-le en quelquechose.html sur ton ordinateur — 4. Double-clique pour l'ouvrir dans ton navigateur. C'est fait !

Tu peux partager le fichier avec n'importe qui — ils n'ont qu'à l'ouvrir dans leur navigateur. Pas d'applis à installer, pas de comptes à créer. Ce site que tu lis en ce moment est un fichier HTML créé avec l'IA.

Quand utiliser quoi

Utilise Markdown

Dissertations, rapports, notes, guides d'étude, emails, documents que tu colleras dans Google Docs ou Word

Utilise HTML

Quiz, présentations interactives, jeux, graphiques, sites web, tout ce que les autres verront dans un navigateur

Astuce Pro

Tu n'as pas besoin d'apprendre ces formats. Dis simplement à l'IA ce que tu veux et ajoute « en Markdown » ou « en fichier HTML » — l'IA fait le reste !

Point clé

Plus tu comprends ces formats, plus tes résultats avec l'IA seront puissants. C'est une question de communication, pas de programmation.

Pourquoi Mermaid ?

Mermaid prend en charge les organigrammes, diagrammes de séquence, diagrammes de Gantt, cartes mentales, chronologies, camemberts, diagrammes de classes et plus encore. Publier gratuitement — GitHub Pages

Tu as créé une appli interactive, un jeu ou un site web incroyable avec l'IA. Et maintenant ? Partage-le avec le monde entier gratuitement grâce à GitHub Pages. Toute personne ayant le lien peut utiliser ta création — pas de frais d'hébergement, pas de serveur à configurer, pas de connaissances techniques nécessaires.

Étape 1 : Créer un compte GitHub gratuit

Va sur github.com et inscris-toi gratuitement. C'est ta plateforme pour publier des sites web et partager du code.

Étape 2 : Créer un nouveau dépôt (repository)

Clique sur « New repository », donne-lui un nom (il fera partie de ton URL), et assure-toi qu'il est en Public. Un dépôt, c'est comme un dossier pour ton projet.

Étape 3 : Télécharger ton fichier HTML

Clique sur « Add file » → « Upload files » et glisse ton fichier HTML. Important : nomme ton fichier principal index.html — c'est ce que GitHub Pages affichera quand quelqu'un visitera ton site.

Étape 4 : Activer GitHub Pages

Va dans Settings → Pages de ton dépôt. Sous « Source », sélectionne « Deploy from a branch », choisis la branche « main », et clique sur Save. En quelques secondes, ton site est en ligne !

Étape 5 : Partager ton lien

Ton site sera accessible à https://tonnomdutilisateur.github.io/nom-du-depot/. Partage ce lien avec n'importe qui — ils pourront ouvrir ta création interactive instantanément, sur n'importe quel appareil. Ce site que tu lis en ce moment est hébergé sur GitHub Pages !

100% Gratuit

Pas de frais d'hébergement, pas de carte de crédit, pas de limites pour les projets personnels. GitHub Pages est entièrement gratuit.

Mises à jour instantanées

Télécharge une nouvelle version de ton fichier et le site se met à jour automatiquement. Itère rapidement avec l'IA.

Partage tout

Jeux, quiz, portfolios, histoires interactives, visualisations de données — si l'IA peut le créer, GitHub Pages peut l'héberger.

Astuce Pro

Tu peux même utiliser un nom de domaine personnalisé ! GitHub Pages prend en charge le HTTPS gratuit et les domaines personnalisés pour un look professionnel.

Méga Conseils

Les directives essentielles pour obtenir les meilleurs résultats avec l'IA

Le conseil le plus important de tous

Demande à l'IA de te poser des questions d'abord

« Avant de répondre, pose-moi des questions pour clarifier ta compréhension de ma demande. »

Cette seule phrase améliore énormément la qualité des résultats de l'IA. Au lieu de deviner ce que tu veux, l'IA te posera des questions de clarification intelligentes — comme le ferait un excellent collaborateur humain. Le résultat est toujours meilleur quand l'IA comprend vraiment ton intention.

Directives pour parler à l'IA

Mieux tu communiques avec l'IA, meilleurs sont les résultats. Voici les ingrédients clés pour de superbes conversations avec l'IA :

1. Donne du contexte

Explique le contexte à l'IA. Quelle est la situation ? Que sais-tu déjà ? Qu'as-tu essayé ? Le contexte aide l'IA à donner des réponses pertinentes et utiles plutôt que génériques.

2. Personnalise

Rends-le personnel avec ton expérience. Partage ta perspective, ton style, ta voix. Le résultat doit te ressembler, pas un texte d'IA générique. Rends-le authentique pour les autres.

3. Définis ton public

Qui verra le résultat ? Des ados ? Des profs ? Des investisseurs ? Un ami ? Le public change tout — le langage, le ton, le niveau de détail, les exemples.

4. Explique ton objectif

Qu'essaies-tu d'accomplir ? Pas juste « écris une dissertation » mais « convaincs ma classe que les énergies renouvelables sont urgentes ». Plus l'objectif est clair, plus le résultat est précis.

5. Décris le résultat souhaité

À quoi ça doit ressembler ? Quelle longueur ? Quel format ? Quelle structure ? « Donne-moi 5 points » vs. « Écris une dissertation de 500 mots » produit des résultats très différents.

6. Décris l'ambiance

Quel est le look, le ressenti, le style ? Fun et ludique ? Sérieux et professionnel ? Minimaliste et épuré ? L'ambiance façonne tout, du choix des mots au design visuel.

7. Décris l'impact

Quelle émotion veux-tu susciter ? Quelle réaction veux-tu provoquer ? « Je veux que les gens se sentent inspirés et passent à l'action » guide l'IA de manière puissante.

8. Fournis des références

Partage des exemples de ce que tu aimes et n'aimes pas. « Fais-le comme ce site » ou « J'adore ce style mais je déteste les couleurs ». Les références valent mille mots.

Le Mégaconseil pour le contenu interactif

Demande un seul fichier HTML avec tout intégré

Quand tu demandes à l'IA de créer du contenu interactif — un quiz, un jeu, un tableau de bord, une présentation interactive — utilise cette phrase magique :

« Crée un seul fichier HTML avec tout le code JavaScript, CSS et HTML intégré dedans. »

Cela crée un fichier « tout-en-un » qui :

  • Fonctionne localement sur n'importe quel ordinateur — double-clique pour l'ouvrir
  • N'a besoin d'aucun logiciel spécial, pas d'installations, pas de dépendances
  • Peut être partagé avec n'importe qui — ils n'ont qu'à l'ouvrir dans leur navigateur
  • Ce site même que tu lis en ce moment a été construit exactement de cette façon !

Bonus : Rends-le encore mieux avec des bibliothèques cloud

Tu veux des graphiques plus élaborés, des animations ou des éléments d'interface plus sophistiqués ? Demande à l'IA :

« Y a-t-il une bibliothèque qui peut rendre ça plus interactif ? Utilise un lien CDN pour que ça marche sans rien télécharger. »

Les bibliothèques CDN (Content Delivery Network) se chargent depuis internet, donc ton fichier HTML reste petit et simple. Le seul compromis : le fichier a besoin d'une connexion internet pour fonctionner. Mais ça vaut le coup pour de beaux graphiques, des animations fluides et des interfaces soignées.
