Markdown — ドキュメントとテキスト用

Markdownはテキストをフォーマットする非常にシンプルな方法です。AIが応答をフォーマットするために使用しているもの — 見出し、太字テキスト、箇条書きリスト、表です。クリーンで、不要なフォーマットがなく、読みやすく編集しやすいです。AIにドキュメントを作成してもらうときは、Markdown形式を要求してください — そしてそれをGoogle DocsやWord文書に貼り付けます。

Google DocsでMarkdownを使う方法

ステップ1: AIにMarkdownを求める

AIにドキュメントを書いてもらうときは、追加してください：「Markdown形式で出力してください」。結果は、混乱したフォーマットのない、クリーンで構造化されたテキストになります。

ステップ2: Google Docsに貼り付ける

最良の方法: Google Workspace Marketplaceから「Docs to Markdown」アドオンをインストールします。ワンクリックでMarkdownを完璧にフォーマットされたGoogle Docsテキストに変換します。

クイック方法: Markdownをコピーして貼り付け、Ctrl+Shift+V（フォーマットなしで貼り付け）を使用します。見出しとリストがクリーンで簡単にフォーマットできます。

ステップ3: Google DocsをMarkdownとしてダウンロード

Google Docsは直接Markdownにエクスポートできます！ファイル → ダウンロード → Markdown (.md)に移動します。これにより、AIにフィードバックしたり、GitHubで使用したり、他の形式に変換できるクリーンなMarkdownファイルが得られます。逆も同様です — .mdファイルをGoogle Docsで開くと、完全なフォーマットでレンダリングされます。

なぜ通常のテキストではなくMarkdownなのか？

Markdownはクリーンです — コピー＆ペーストで壊れる隠しフォーマットがありません。普遍的です — どこでも機能します。そしてAIの考え方です — AIの言語で話すと、より良い結果が得られます。

HTML — インタラクティブコンテンツ用

人々がブラウザで開いて対話できるもの — ウェブサイト、クイズ、インタラクティブチャート、プレゼンテーションなど — を作成したいときは、AIにHTMLを提供してもらいます。HTMLを学ぶ必要はありません。 AIがすべてのコードを書きます。ファイルを保存してChromeまたは任意のブラウザで開くだけです。

AIに依頼するもの

「フランス革命についての10問の インタラクティブクイズを作成して ください。スコアトラッカーと カラフルなデザイン付きです。 ブラウザで開ける単一のHTML ファイルをください。」 AIはすべてのコードを生成します： - HTML（構造） - CSS（デザイン） - JavaScript（インタラクティビティ） .htmlファイルとして保存して 開くだけです！それだけです。

Mermaid — テキストからダイアグラム

Mermaidを使用すると、プロフェッショナルなダイアグラム — フローチャート、タイムライン、マインドマップ、シーケンス図 — をシンプルなテキストを書くだけで作成できます。AIがMermaidコードを書きます。 ダイアグラムを説明し、コードをMermaid Live Editorに貼り付けてエクスポートするだけです。Mermaidを使用して、複雑なフローやプロセスをAIに説明することもできます。

なぜMermaid？

Mermaidはフローチャート、シーケンス図、ガントチャート、マインドマップ、タイムライン、円グラフ、クラス図などをサポートします。コードはテキストベースなので、AIが簡単に生成および変更できます。GitHubはMarkdownファイルでMermaidをネイティブにレンダリングし、Live EditorではPNGまたはSVGとしてエクスポートできます。

プロのヒント: Mermaidを使用して、複雑なフローをAIに説明することもできます — Mermaid図をプロンプトに貼り付けて、AIがあなたのプロセスやアーキテクチャを一目で理解できるようにします。

無料で公開 — GitHub Pages

AIで素晴らしいインタラクティブアプリ、ゲーム、またはウェブサイトを作成しました。次はどうする？GitHub Pagesを使用して世界に無料で共有します。リンクを持っている人は誰でもあなたの作品を使用できます — ホスティング料金なし、サーバーセットアップなし、技術的な知識不要です。

ステップ1: 無料のGitHubアカウントを作成

github.comにアクセスし、無料アカウントにサインアップします。これがウェブサイトを公開しコードを共有するためのホームです。

ステップ2: 新しいリポジトリを作成

「新しいリポジトリ」をクリックし、名前を付け（これがURLの一部になります）、公開に設定されていることを確認します。リポジトリはプロジェクトのフォルダと考えてください。

ステップ3: HTMLファイルをアップロード

「ファイルを追加」→「ファイルをアップロード」をクリックし、HTMLファイルをドラッグします。重要: メインファイルの名前をindex.htmlにします — これは誰かがあなたのサイトを訪問したときにGitHub Pagesが表示するものです。

ステップ4: GitHub Pagesを有効化

リポジトリの設定→Pagesに移動します。「ソース」の下で、「ブランチからデプロイ」を選択し、「main」ブランチを選択して保存をクリックします。数秒後、サイトが公開されます！

ステップ5: リンクを共有

サイトはhttps://yourusername.github.io/your-repo-name/で公開されます。このリンクを誰とでも共有してください — 任意のデバイスでインタラクティブな作品をすぐに開くことができます。まさにこのウェブサイトはGitHub Pagesでホストされています！

100%無料

ホスティング料金なし、クレジットカード不要、個人プロジェクトに制限なし。GitHub Pagesは完全に無料です。

即座の更新

新しいバージョンのファイルをアップロードすると、サイトは自動的に更新されます。AIで高速に反復できます。

何でも共有

ゲーム、クイズ、ポートフォリオ、インタラクティブストーリー、データ可視化 — AIが構築できるものなら、GitHub Pagesがホストできます。

プロのヒント

カスタムドメイン名も使用できます！GitHub Pagesは無料のHTTPSとカスタムドメインをサポートし、プロフェッショナルな印象を与えます。
