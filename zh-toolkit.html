Markdown — 用于文档和文本

Markdown是一种超级简单的文本格式化方式。这是AI用来格式化其响应的方式 — 标题、粗体文本、项目列表、表格。它简洁、没有不必要的格式，易于阅读和编辑。当你让AI创建文档时，请求Markdown格式 — 然后粘贴到Google Docs或你的文字处理器中。

如何在Google Docs中使用Markdown

步骤0：准备

首先，在Google Docs中启用原生Markdown支持：进入工具 → 偏好设置并勾选“启用Markdown”。这将解锁从Markdown粘贴和自动Markdown格式化等功能。

如需高级Markdown使用，可以选择从Google Workspace Marketplace安装Docs to Markdown插件。但是，原生Markdown支持对于大多数用户和我们的场景已经足够了。

步骤1：向AI请求Markdown

当要求AI编写文档时，添加：“请以Markdown格式输出”。结果将是干净、结构良好的文本，没有任何混乱的格式。

步骤2：粘贴到Google Docs

从AI工具复制Markdown文本，然后在Google Docs中使用编辑 → 从Markdown粘贴（或右键选择“从Markdown粘贴”）。Google Docs将自动将Markdown转换为美观的格式化文本 — 标题、加粗、列表、链接等都会正确显示。

注意：从Markdown粘贴选项仅在完成步骤0（在偏好设置中启用Markdown）后才可用。如果看不到该选项，请返回工具 → 偏好设置确认“启用Markdown”已勾选。

步骤3：将Google Docs下载为Markdown

Google Docs可以直接导出为Markdown！转到文件 → 下载 → Markdown (.md)。这为你提供了一个干净的Markdown文件，你可以反馈给AI，在GitHub中使用，或转换为任何其他格式。它也可以反向工作 — 你可以在Google Docs中打开.md文件，它将以完整格式呈现。

为什么用Markdown而不是普通文本？

Markdown很干净 — 没有复制粘贴时会破坏的隐藏格式。它是通用的 — 到处都能用。而且这是AI的思维方式 — 当你说它的语言时，你会得到更好的结果。

免费发布 — GitHub Pages

你已经用AI创建了一个令人惊叹的交互式应用程序、游戏或网站。现在怎么办？使用GitHub Pages免费与世界分享它。任何有链接的人都可以使用你的创作 — 无托管费用，无服务器设置，不需要技术知识。

步骤1：创建免费GitHub账户

前往github.com并注册一个免费账户。这是你发布网站和分享代码的家。

步骤2：创建新仓库

点击"新建仓库"，给它一个名称（这将成为你的URL的一部分），并确保它设置为公开。将仓库视为项目的文件夹。

步骤3：上传你的HTML文件

点击"添加文件" → "上传文件"并将你的HTML文件拖入。重要：将你的主文件命名为index.html — 这是当有人访问你的网站时GitHub Pages将显示的内容。

步骤4：启用GitHub Pages

转到你的仓库的设置 → Pages。在"源"下，选择"从分支部署"，选择"main"分支，然后点击保存。几秒钟后，你的网站就上线了！

步骤5：分享你的链接

你的网站将在https://yourusername.github.io/your-repo-name/上线。与任何人分享此链接 — 他们可以在任何设备上即时打开你的交互式创作。这个网站就托管在GitHub Pages上！
