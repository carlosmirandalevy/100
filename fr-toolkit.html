Markdown — Pour les documents & le texte

Markdown est un moyen ultra simple de formater du texte. C'est ce que l'IA utilise pour formater ses réponses — titres, texte en gras, listes à puces, tableaux. C'est propre, sans formatage inutile, et facile à lire et modifier. Quand tu demandes à l'IA de créer des documents, demande le format Markdown — puis colle-le dans Google Docs ou ton traitement de texte.

Comment utiliser Markdown avec Google Docs

Étape 0 : Préparation

D'abord, active la prise en charge native de Markdown dans Google Docs : va dans Outils → Préférences et coche « Activer Markdown ». Cela débloque des fonctions comme Coller depuis Markdown et le formatage automatique Markdown.

Pour une utilisation avancée de Markdown, tu peux optionnellement installer le plugin Docs to Markdown depuis le Google Workspace Marketplace. Cependant, le support natif de Markdown est largement suffisant pour la plupart des utilisateurs et nos scénarios.

Étape 1 : Demande du Markdown à l'IA

Quand tu demandes à l'IA d'écrire un document, ajoute : « Donne-moi le résultat en format Markdown ». Le résultat sera un texte propre et bien structuré, sans formatage inutile.

Étape 2 : Coller dans Google Docs

Copie le texte Markdown depuis ton outil d'IA, puis dans Google Docs utilise Édition → Coller depuis Markdown (ou clic droit et sélectionne « Coller depuis Markdown »). Google Docs convertira automatiquement le Markdown en texte magnifiquement formaté — titres, gras, listes, liens et plus encore seront correctement rendus.

Note : L'option Coller depuis Markdown n'est disponible que si tu as complété l'étape 0 (activation de Markdown dans les Préférences). Si tu ne la vois pas, retourne dans Outils → Préférences et vérifie que « Activer Markdown » est coché.

Étape 3 : Télécharger Google Docs en Markdown

Google Docs peut exporter directement en Markdown ! Va dans Fichier → Télécharger → Markdown (.md). Tu obtiens un fichier Markdown propre que tu peux donner à l'IA, utiliser sur GitHub, ou convertir dans n'importe quel format. Ça marche aussi dans l'autre sens — tu peux ouvrir un fichier .md dans Google Docs et il s'affichera avec tout le formatage.

Pourquoi Markdown plutôt que du texte normal ?

Markdown est propre — pas de formatage caché qui casse quand tu copies-colles. C'est universel — ça marche partout. Et c'est le langage de l'IA — tu obtiendras de meilleurs résultats en parlant sa langue.

Publier gratuitement — GitHub Pages

Tu as créé une appli interactive, un jeu ou un site web incroyable avec l'IA. Et maintenant ? Partage-le avec le monde entier gratuitement grâce à GitHub Pages. Toute personne ayant le lien peut utiliser ta création — pas de frais d'hébergement, pas de serveur à configurer, pas de connaissances techniques nécessaires.

Étape 1 : Créer un compte GitHub gratuit

Va sur github.com et inscris-toi gratuitement. C'est ta plateforme pour publier des sites web et partager du code.

Étape 2 : Créer un nouveau dépôt (repository)

Clique sur « New repository », donne-lui un nom (il fera partie de ton URL), et assure-toi qu'il est en Public. Un dépôt, c'est comme un dossier pour ton projet.

Étape 3 : Télécharger ton fichier HTML

Clique sur « Add file » → « Upload files » et glisse ton fichier HTML. Important : nomme ton fichier principal index.html — c'est ce que GitHub Pages affichera quand quelqu'un visitera ton site.

Étape 4 : Activer GitHub Pages

Va dans Settings → Pages de ton dépôt. Sous « Source », sélectionne « Deploy from a branch », choisis la branche « main », et clique sur Save. En quelques secondes, ton site est en ligne !

Étape 5 : Partager ton lien

Ton site sera accessible à https://tonnomdutilisateur.github.io/nom-du-depot/. Partage ce lien avec n'importe qui — ils pourront ouvrir ta création interactive instantanément, sur n'importe quel appareil. Ce site que tu lis en ce moment est hébergé sur GitHub Pages !

100% Gratuit

Pas de frais d'hébergement, pas de carte de crédit, pas de limites pour les projets personnels. GitHub Pages est entièrement gratuit.

Mises à jour instantanées

Télécharge une nouvelle version de ton fichier et le site se met à jour automatiquement. Itère rapidement avec l'IA.

Partage tout

Jeux, quiz, portfolios, histoires interactives, visualisations de données — si l'IA peut le créer, GitHub Pages peut l'héberger.

Astuce Pro

Tu peux même utiliser un nom de domaine personnalisé ! GitHub Pages prend en charge le HTTPS gratuit et les domaines personnalisés pour un look professionnel.
