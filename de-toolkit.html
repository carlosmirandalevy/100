Markdown — Für Dokumente & Text

Markdown ist eine super einfache Art, Text zu formatieren. KI nutzt es, um ihre Antworten zu formatieren — Überschriften, fetter Text, Aufzählungen, Tabellen. Es ist sauber, frei von unnötiger Formatierung und einfach zu lesen und zu bearbeiten. Wenn du KI bittest, Dokumente zu erstellen, frage nach Markdown-Format — dann füge es in Google Docs oder dein Textverarbeitungsprogramm ein.

Wie man Markdown mit Google Docs nutzt

Schritt 0: Vorbereitung

Aktiviere zuerst die native Markdown-Unterstützung in Google Docs: gehe zu Tools → Einstellungen und aktiviere „Markdown aktivieren“. Das schaltet Funktionen wie Aus Markdown einfügen und automatische Markdown-Formatierung frei.

Für fortgeschrittene Markdown-Nutzung kannst du optional das Plugin Docs to Markdown aus dem Google Workspace Marketplace installieren. Die native Markdown-Unterstützung reicht jedoch für die meisten Nutzer und unsere Szenarien völlig aus.

Schritt 1: Bitte KI um Markdown

Wenn du KI bittest, ein Dokument zu schreiben, füge hinzu: "Bitte gib es im Markdown-Format aus". Das Ergebnis wird sauberer, gut strukturierter Text ohne unordentliche Formatierung sein.

Schritt 2: In Google Docs einfügen

Kopiere den Markdown-Text aus deinem KI-Tool, dann nutze in Google Docs Bearbeiten → Aus Markdown einfügen (oder Rechtsklick und „Aus Markdown einfügen“ wählen). Google Docs konvertiert das Markdown automatisch in schön formatierten Text — Überschriften, Fettdruck, Listen, Links und mehr werden korrekt dargestellt.

Hinweis: Die Option Aus Markdown einfügen ist nur verfügbar, wenn du Schritt 0 abgeschlossen hast (Markdown in den Einstellungen aktiviert). Falls du sie nicht siehst, gehe zurück zu Tools → Einstellungen und stelle sicher, dass „Markdown aktivieren“ aktiviert ist.

Schritt 3: Google Docs als Markdown herunterladen

Google Docs kann direkt nach Markdown exportieren! Gehe zu Datei → Herunterladen → Markdown (.md). Das gibt dir eine saubere Markdown-Datei, die du zurück zur KI geben, in GitHub nutzen oder in jedes andere Format konvertieren kannst. Es funktioniert auch umgekehrt — du kannst eine .md-Datei in Google Docs öffnen und sie wird mit voller Formatierung dargestellt.

Warum Markdown statt normalem Text?

Markdown ist sauber — keine versteckte Formatierung, die beim Kopieren-Einfügen kaputt geht. Es ist universal — funktioniert überall. Und es ist wie KI denkt — du bekommst bessere Ergebnisse, wenn du ihre Sprache sprichst.

HTML — Für interaktive Inhalte

Wenn du etwas erstellen möchtest, das Menschen in einem Browser öffnen und damit interagieren können — wie eine Website, ein Quiz, ein interaktives Diagramm oder eine Präsentation — bitte KI, dir HTML zu geben. Du musst HTML nicht lernen. Die KI schreibt den gesamten Code für dich. Du speicherst einfach die Datei und öffnest sie in Chrome oder einem beliebigen Browser.

Was du die KI fragst

"Erstelle ein interaktives Quiz über die Französische Revolution mit 10 Fragen, einem Punktezähler und farbenfrohem Design. Gib mir eine einzelne HTML-Datei, die ich in einem Browser öffnen kann." Die KI generiert ALLEN Code: - HTML (die Struktur) - CSS (das Design) - JavaScript (die Interaktivität) Du speicherst es einfach als .html-Datei und öffnest es! Das war's.

Wie es funktioniert — Es ist wirklich so einfach

1. Bitte KI, alles zu erstellen, was du willst, als "eine einzelne HTML-Datei" — 2. Kopiere den Code, den dir die KI gibt — 3. Speichere ihn als etwas.html auf deinem Computer — 4. Doppelklicke, um es in deinem Browser zu öffnen. Fertig!

Du kannst die Datei teilen mit jedem — sie öffnen sie einfach in ihrem Browser. Keine Apps zum Installieren, keine Konten zum Erstellen. Diese Website, die du gerade liest, ist eine HTML-Datei, die mit KI erstellt wurde.

Kostenlos veröffentlichen — GitHub Pages

Du hast eine erstaunliche interaktive App, ein Spiel oder eine Website mit KI erstellt. Und jetzt? Teile sie kostenlos mit der Welt mit GitHub Pages. Jeder mit dem Link kann deine Kreation nutzen — keine Hosting-Gebühren, kein Server-Setup, kein technisches Wissen nötig.

Schritt 1: Kostenloses GitHub-Konto erstellen

Gehe zu github.com und melde dich kostenlos an. Das ist dein Zuhause für die Veröffentlichung von Websites und das Teilen von Code.

Schritt 2: Neues Repository erstellen

Klicke auf "New repository", gib ihm einen Namen (dieser wird Teil deiner URL) und stelle sicher, dass es auf Public gesetzt ist. Denke an ein Repository als einen Ordner für dein Projekt.

Schritt 3: HTML-Datei hochladen

Klicke auf "Add file" → "Upload files" und ziehe deine HTML-Datei hinein. Wichtig: Benenne deine Hauptdatei index.html — das ist was GitHub Pages zeigt, wenn jemand deine Seite besucht.

Schritt 4: GitHub Pages aktivieren

Gehe zu den Settings → Pages deines Repositorys. Unter "Source", wähle "Deploy from a branch", wähle "main" Branch und klicke Save. In ein paar Sekunden ist deine Seite live!

Schritt 5: Link teilen

Deine Seite wird live sein unter https://deinbenutzername.github.io/dein-repo-name/. Teile diesen Link mit jedem — sie können deine interaktive Kreation sofort öffnen, auf jedem Gerät. Diese Website ist auf GitHub Pages gehostet!

100% Kostenlos

Keine Hosting-Gebühren, keine Kreditkarte, keine Limits für persönliche Projekte. GitHub Pages ist komplett kostenlos.

Sofortige Updates

Lade eine neue Version deiner Datei hoch und die Seite aktualisiert sich automatisch. Iteriere schnell mit KI.

Alles teilen

Spiele, Quizze, Portfolios, interaktive Geschichten, Datenvisualisierungen — wenn KI es bauen kann, kann GitHub Pages es hosten.

Profi-Tipp

Du kannst sogar einen benutzerdefinierten Domainnamen verwenden! GitHub Pages unterstützt kostenloses HTTPS und benutzerdefinierte Domains für einen professionellen Touch.
